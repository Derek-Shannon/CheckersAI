# Checkers4All!

A classic game of **Checkers** implemented using Python's **Pygame** library, featuring a challenging AI opponent powered by the **Minimax algorithm**.

## Features

* **Player vs. AI (PvAI):** Challenge the Minimax-based AI (playing as **Black**).
* **Player vs. Player (PvP):** Play a local match against a friend.
* **Standard Checkers Rules:** Includes **mandatory jumps** and **kinging** mechanics.
* **Intuitive UI:** Clear turn indicators, valid move highlights (green circles), and a piece-taken counter.
* **Modular Scene Management:** Easily switch between the **Main Menu**, **Game**, and **Game Over** scenes.

## Installation & Setup

### Prerequisites

    You need at least **Python 3** installed on your system. and pip
    pip should come pre installed with python 3. If you do not have it, please redownload Python 3 and check to make sure that pip is checked in the extra install options.



### Install pygame via command prompt (on windows)

    pip install pygame

### If that does not work, try

    py pip install pygame

### How to launch

*    launch **game.py** through command prompt or by double clicking **game.py**
